DESAIN ANTARMUKA PENGGUNA

Yustika Dewi Amelia

Deskripsi

Implementasi Desain Antarmuka Pengguna merupakan matakuliah pada kurikulum 2016 di prodi Diploma 3 Sistem Informasi semester 2. Mata kuliah ini memperlajari Prinsip Dasar User Interface Design, User Profile & User Centered Design.
Selain itu pada mata kuliah ini akan mempelajari web desain dengan materi Dasar HTML, Tabel, Form & HTML 5 kemudian memberikan style dengan menggunakan CSS dan diakhir mata kuliah akan diajarkan mengenai konsep dan implementasi dari Web Responsive.
